Airlines && Hotels Search Website

This is a Database Design && Web Development project. The purpose is to build a website that can help people easily find their airlines and the high Price-performance ratio hotels.

  • Using HTML/CSS/Boostrap/FontAwesome to build the front-end work and Angular as a main helper.
  • Using NodeJS as the server and Mongoose to connect to the MongoDB.
  • Using ExpressJS to handle the HTTP request and response from the server part.
  • Create 4 schemas for Mongoose (User, Hotel, Flight, Contact). It can spread more schemas by following the first normal form.
  • Using Openshit to deploy the nodeJS work.
  • Using third party API (FlightsStates && Amadeus) to get the accurate information from the database.
  • The website uses responsive design and can fit for different sizes of devices.

weather Searching System

The purpose of the project is to leverage a weather API (https://developer.forecast.io) combined with a geolocation API (https://developers.google.com/maps) to allow users to retrieve current as well as historic data about any location that the users search.

  • The website can be able to enter in any location and retrieve current and useful information about the weather in that area, as well as a future forecast.
  • The website also include charts for historic weather information about that area using google visualization library and allowing the user to see relevant information within a reasonable time period.
  • The website allows a user to track his or her search history stored in sessions.
